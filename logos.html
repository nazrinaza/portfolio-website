How to Use

Export & Implementation

📥

Download SVG

Right-click on any logo → "Save as" → Choose SVG format. Vector files scale infinitely without quality loss — perfect for any size application.

🖼️

Convert to PNG

Use an online SVG-to-PNG converter (like CloudConvert or SVGExport) for raster formats. Export at 2x or 3x size for retina displays.

💻

Embed in HTML

Copy the SVG code directly and paste it into your HTML. Inline SVGs are searchable, accessible, and can be styled with CSS.

🎨

Customize Colors

SVG colors are easily editable. Change the gradient stops or solid colors to match your brand palette using any text editor.
